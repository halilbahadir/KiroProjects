# E-Commerce Website

A simple 3-tier e-commerce application with React frontend, Node.js backend, and SQLite database.

## Setup

### Backend
```bash
cd backend
npm install
npm start
```
Server runs on http://localhost:3001

### Frontend
```bash
cd frontend
npm install
npm start
```
App runs on http://localhost:3000

## Features

- Home page with 20 products
- Product detail page with reviews
- Shopping cart with add/update/delete functionality
- SQLite database for persistence

## Architecture

- **Frontend**: React with React Router
- **Backend**: Express.js REST API
- **Database**: SQLite with products and cart tables
